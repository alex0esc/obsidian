- Unmittelbare Adressierung = Adresse ist teil des Befehls, nicht möglich in Risc V (keine variable Befehlsgröße)
- Register-Adressierung = Wert wird sich aus Register geladen
- Effektive Adresse = Dynamische Adressen die zur Laufzeit berechnet wird 7 => Rgisteradresse + Offset
=> Mehr Adressierungsarte steigern die Komplexität


- Dreiadressform (Risc V)
- ZweiAdressform (x86)
- Einadressform (Nicht mehr verwendet, nur in 8bit prozessoren) mit Akkumulator
- Nulladressform (Arbeitet nur mit stack,  simpel, aber viele Speicherzugriffe auf den stack, Java Virtual Machine)
